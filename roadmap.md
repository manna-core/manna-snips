# Manna Snips Roadmap

## Mission

Ship `Manna Snips` as a small Windows screenshot utility that is faster and more honest than the default save-heavy workflow:

- copy first
- download explicitly
- stay local
- stay simple

## Current Phase

Phase 1: Standalone repo and first public release prep

## Phase Plan

### Phase 1 - Standalone Shipping Surface

- move from root-script utility into a standalone repo surface
- harden local paths for real user machines
- add packaging, README, and release notes
- keep the main screen obvious and low-friction

### Phase 2 - Release Hardening

- configurable hotkeys
- startup or tray decision
- cleaner release assets and screenshots
- optional first GitHub release

### Phase 3 - Small Utility Refinement

- selective editor improvements only if they stay fast
- more release automation if the first public version lands well
